Algorithm for Fact-Checking Tool

Our Algorithm uses the Google Fact Check Tool API and information from the Database of Domain Big Data in order to provide the credibility of a fact and the credentials of a website. The information is purely true with no false facts.

ONE

Google Fact Check API

API containing data from the database of Google Fact Check Tools is used to check credibility of facts.


TWO

Web Scraping

When a Fact is entered, the credibility of top 10 websites on Google is provided so that the user knows what is correct and what is not.

THREE

Domain Big Data

The credentials of a website, that is the name of owner, website age,etc is extracted from the database of DOmain Big Data


FOUR

Machine Learning

The Fact Checking Machine Learning Model uses sklearn library and ML algorithms so predict the credibility of a website.
